# 堆

Portainer中的一个堆对应于由一个`docker-compose.yaml`文件管理的所有服务. 

!!! tip

    `docker-compose.yaml`文件中的相对路径中`.`表示的是Host下的`/data`文件夹. 绝对路径表示的就是Host下的绝对路径. 企业版本应该会提供`.`映射到其他路径的功能.
