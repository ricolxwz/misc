# 常用配置

## admin

admin就是第一次登录的时候创建的用户.

::: tip
可以通过修改`app.ini`文件的`[service]`项的`DISABLE_REGISTRATION`禁用注册.
:::

## 主页

可以通过修改`app.ini`文件的`[server]`项的`LANDIING_PAGE`修改默认主页.
